
# Gesture-Triggered Meme Generator 

An interactive, browser-based application that uses your webcam and computer vision to track your body, hands, and face, triggering different meme images based on the specific physical gestures you make!

Built with Vanilla JavaScript and **Google MediaPipe Holistic**, this project draws a real-time, futuristic blue wireframe over your body and acts as a custom gesture-recognition engine.

##  Prerequisites

* A modern web browser (Google Chrome, Edge, Firefox, or Safari).
* A working webcam.
* Local server environment (optional, but recommended, like VS Code's "Live Server" extension).

##  How to Clone and Set Up

**1. Clone the repository**
Open your terminal or command prompt and run the following command to clone the project to your local machine:

```bash
git clone https://github.com/TheSkyroo/fahhhhh.git

```

**2. Navigate into the project folder**



##  How to Use

1. Open the `index.html` file in your web browser. You can do this by double-clicking the file, dragging it into an open browser window, or using a local server (like Live Server in VS Code).
2. Your browser will ask for **Camera Permissions**. Click **Allow**.
3. Stand/sit far back enough so the camera can see your face, chest, and hands.
4. Once the blue wireframe appears on your body, start performing the gestures listed above to trigger your memes!

##  Tech Stack

* HTML5 / CSS3
* Vanilla JavaScript
* [MediaPipe Holistic](https://www.google.com/search?q=https://google.github.io/mediapipe/solutions/holistic.html) (via CDN)
